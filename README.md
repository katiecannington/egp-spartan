# Spartan Plumbing

Generated by EGP Website Builder.

## Deploy

1. `npm install`
2. `npm run dev` (local) or push to GitHub + connect Vercel
3. Edit `src/config.js` to update content

## Structure

- `src/config.js` — All content (edit this)
- `src/App.jsx` — Shared layout
- `src/pages/` — Individual pages
- `public/egp-logo-white.png` — EGP branding (do not remove)

## Important

Make sure `public/egp-logo-white.png` exists — it shows in the footer.
If missing, add the white EGP logo to the `public/` folder.
